About Cellcom Order Tracker

Description

Cellcom Order Tracker is an internal web application designed to help Cellcom store representatives and managers track mobile phone orders from initial creation through activation and beyond. The system manages customer information, mobile device inventory, Bell rate plans, and the complete order lifecycle. It provides a centralized platform for order tracking, status management, and customer relationship management across all Cellcom store locations.

Current Features

  • Multi-user login system (Anthony, Dominic, Rene, Admin)
  • Order tracking across full lifecycle (New ‚Üí Pending Activation ‚Üí Activated ‚Üí Cancelled/Returned)
  • Linked entities: customer, phone, rate plan on each order
  • Basic filters for status, owner, store location
  • Phone catalog with 20+ Bell devices (brands, models, storage, colours, pricing)
  • Bell rate plan reference catalog
  • Customer management with search functionality
  • Order status history timeline
  • Responsive, professional UI with Cellcom-inspired branding

Future Features / Roadmap

  • SMS/Email Updates: Automated notifications to customers on status changes
  • REST APIs/Webhooks: Integration endpoints to connect with Bell and third-party systems
  • Analytics Dashboard: Orders per store/rep, rate plan mix, activation times, performance metrics
  • Multi-Carrier Support: Expand beyond Bell to support Telus, Rogers, Fizz, and other carriers
  • Store/Region Hierarchy: Advanced permissioning and organizational structure
  • Inventory Integration: Real-time stock availability per store location
  • Customer Portal: Self-service order tracking for customers
  • Reporting: Export orders to CSV/PDF, generate monthly reports
  • Mobile App: Native iOS/Android app for reps on the go

Technical Stack

  • Backend: Python 3, Flask, SQLAlchemy
  • Frontend: Jinja2 templates, vanilla JavaScript, custom CSS
  • Database: SQLite (dev), MySQL/PostgreSQL (production)
  • Authentication: Flask sessions with bcrypt password hashing
  • Deployment: Gunicorn WSGI server, Nginx reverse proxy, HostPapa VPS
